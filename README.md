# NyxNight

NyxNight is a standalone local-first night-out planner generated by the repository's Google ADK factory workflow.

Important architecture boundary:

- `factory/` is the Google ADK creator and preserved provenance.
- `nyxnight/` is the generated application.
- The application has no Google ADK runtime dependency and makes no model or network call while shaping a route.
- Each stop provides an explicit user-opened live-search handoff for current places or events; no result is presented as a booked or verified venue.

## Run

```bash
python -m venv .venv
. .venv/bin/activate
pip install -e '.[dev]'
nyxnight doctor
nyxnight serve
```

Open `http://127.0.0.1:8088/`.

## Verify

```bash
./scripts/verify.sh
```

## ADK generation proof

- Actual workflow source: `factory/nyxnight_factory/`
- Immutable blueprint snapshot: `factory/blueprint/`
- Machine-readable run receipt: `docs/adk-generation-receipt.json`

The receipt records the real `SequentialAgent`, `Runner`, session service, stage events, verification output, and generated-file hashes.
